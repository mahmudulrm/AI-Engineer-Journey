# AI Engineer Journey

Welcome to my AI Engineering learning and project repository.

This repository serves as a centralized knowledge base and portfolio documenting my transition from DevOps and Cloud Engineering to AI Engineering.

Topics covered include:

* Python Programming
* FastAPI Development
* Large Language Models (LLMs)
* Prompt Engineering
* Retrieval-Augmented Generation (RAG)
* LangChain and LlamaIndex
* AI Agents and Agentic Workflows
* Voice AI and Speech Technologies
* Azure OpenAI Services
* Automation with n8n
* Vector Databases
* Docker and Kubernetes for AI Applications
* CI/CD for AI Systems
* Monitoring and Observability
* End-to-End AI Projects
* AI Integrator

The repository contains learning notes, code examples, experiments, architecture diagrams, deployment guides, and real-world projects designed to bridge the gap between AI development and production operations.

Current Goal:
Build production-ready AI applications and AI-powered automation systems while applying DevOps best practices for scalability, reliability, and security.
